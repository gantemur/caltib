High-Precision Tibetan Calendrical Math

A rigorous Python library implementing traditional historical algorithms, continuous N-body astronomical reference models, and analytical tools for modern calendar reform.

View Source on GitHub Launch Live Web Calendar →

Historical Accuracy

Exact, faithful implementations of regional Tibetan algorithms, including Phugpa, Tsurphu, Bhutan, Mongol, and Karana traditions.

Astronomical Rigor

Continuous L6 engines backed by truncated ELP2000/82 lunar series and JPL DE422 N-body numerical integration.

Design & Analysis Tools

Built-in CLI tools to generate optimal rational convergents, dyadic approximants, and minimax polynomials for structural reform.

Installation

Install the base package via pip. For advanced diagnostics and continuous ephemeris validation, install the optional dependencies.

# Base installation
pip install caltib

# Include ephemeris data and diagnostic plotting tools
pip install "caltib[ephemeris,diagnostics]"

Quickstart API

The library provides a clean, unified interface to query both discrete traditional engines and continuous astronomical models.

from datetime import date
import caltib

# Evaluate the traditional Phugpa calendar
day_data = caltib.day_info(
    date(2026, 2, 21), 
    engine="phugpa", 
    attributes=("weekday", "sexagenary"),
    debug=True
)

print(day_data)
